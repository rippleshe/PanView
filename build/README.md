# 构建占位图标文件

# 此文件用于占位，实际使用时需要替换为真实的图标文件

# Windows: icon.ico
# macOS: icon.icns
# Linux: icon.png

# 可以使用以下工具创建图标：
# - Windows: GIMP, Photoshop
# - macOS: Icon Composer, Image2icon
# - Linux: GIMP, Inkscape
# - 在线工具: https://www.favicon-generator.org/

# 推荐尺寸：
# - Windows: 256x256
# - macOS: 1024x1024
# - Linux: 512x512
